About this project

This project is mainly focused on the integration of blob style visual elements for the background and the new moder style of webpages like the NFT Art, this project uses the OpenAI API with the ChatGPT 3.5 model for interaction. Due to the specifications of the contest, not being able to use backend programming, it was necessary to use LocalStorage to store the chats.
